# Dockyard
Simple, self-hosted Docker management dashboard with per-user container permissions.
Written in PHP with no framework.

# Features

- Clean list of all your Docker containers
- Start / Stop / View logs
- Per-user permissions: View only, Start, Stop (granular control)
- Admin can force password reset for any user (instantly logs them out)
- Inline container logs with auto-refresh (30s)
- Real-time in-app notifications (new containers, crashes, etc.)

# Using

Clone and Compose, I havent made an image yet.
